# Michael Guo — Portfolio

A minimal, creative portfolio framework for job and internship applications.

## Quick Start

**Preview locally:**

```bash
python -m http.server 8000
# Open http://localhost:8000
```

Or use Live Server in VS Code / Cursor.

## Add Your Resume

1. Copy your resume PDF into the project root
2. Name it `resume.pdf`
3. The "Download Resume (PDF)" button will automatically link to it

## Customize

- **index.html** — Update name, tagline, about, projects, contact email, and social links
- **css/styles.css** — Tweak colors (see `:root` variables) and layout
- **assets/logo.svg** — Replace with your own logo if desired

## Deploy

- **GitHub Pages** — Push to a repo and enable Pages in Settings
- **Netlify** — Drag the folder to netlify.com/drop
- **Vercel** — Connect your repo
