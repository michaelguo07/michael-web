USATT Performance Analyzer

View on Google Chrome Extension Web Store

Overview

Built a Chrome extension that upgrades USA Table Tennis player profiles with automated performance analytics and match insights. It features a custom JavaScript/CSS interface and parses tournament CSV data to compute rating trends, win/loss ratios, opponent statistics, and advanced metrics like notable wins and recent tournament history.
